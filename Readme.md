# Лабораторная работа № 3 - Тру координаты

## Описание

GPS выдает нам координаты в виде широты и долготы. Для того, чтобы рассчитать расстояния между точками часто применяют просто теорему Пифагора, но проблема в том, что это не совсем корректно. Это плюс-минус работает на маленьких расстояниях, но на больших получается ошибка. 

Почему? Да потому Земля - шарообразная (вернее она геоид, и одного чувак сожгли за это, но это сейчас опустим). И один градус широты на экваторе и на полюс это разные расстояния. Надо написать класс, который будет заниматься хранение координат точек, и он должен иметь мтеоды для подсчета расстояния до другой точки, а так же должен быть статический метод, который позволяет рассчитать

## Требования

1. Чистый и понятный код
2. Соблюдение правил форматирования C# кода и проектов
3. Юнит тесты, особенно на пример из описания задачи
4. Класс для хранение координат
5. Работоспособность
